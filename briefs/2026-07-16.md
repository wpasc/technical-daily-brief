### [Linus Torvalds: Linux is not an anti-AI project](https://lore.kernel.org/linux-media/CAHk-=wi4zC+Ze8e+p3tMv8TtG_80KzsZ1syL9anBtmEh5Z40vg@mail.gmail.com/)

tools we use" and that he is willing to put his foot down as top-level
maintainer against blanket anti-AI positions. This effectively settles the
policy question of LLM-assisted contributions for the largest open-source
project in the world; other projects' maintainers will be quoting this thread
in their own debates.

### [A hole in Claude's web_fetch exfiltration defenses](https://simonwillison.net/2026/Jul/15/claude-web-fetch-exfiltration/)
Ayush Paul found a way around the design that is supposed to stop Claude's
web_fetch tool from leaking private data (such as chat memories) to
attacker-controlled URLs; Simon Willison walks through the mechanics. A
concrete reminder that the private-data + attacker-content + egress trifecta
is a structural problem: single-tool mitigations keep getting bypassed.

### [xAI open-sources grok-build after data-upload backlash](https://simonwillison.net/2026/Jul/15/grok-build/)
xAI's grok CLI drew severe criticism after users discovered that running it
could upload the entire working directory -- one user reported SSH keys and a
password manager database -- to xAI's cloud storage. The tool is now open
source, which lets the behavior be audited directly. Worth treating as a case
study in why coding agents need explicit, inspectable upload boundaries.

### [OpenAI's GPT-Red: automated red teaming via self-play](https://openai.com/index/unlocking-self-improvement-gpt-red)
OpenAI describes a system that uses self-play to generate attacks and improve
robustness, including against prompt injection. The interesting claim is the
self-improvement loop for safety properties specifically; the writeup is
official-blog level, so judge the evaluation details before citing numbers.

### [SQLite should have Rust-style editions](https://mort.coffee/home/sqlite-editions/)
An argument that SQLite's extreme backward compatibility freezes old defaults
(the post's examples include type looseness) and that opt-in editions, as Rust
does them, would let the project fix mistakes without breaking existing
databases. Made both HN and Lobsters front pages; the comment threads have
substantive counterpoints from people maintaining long-lived schemas.

### [Firefox compiled to WebAssembly, running in a browser tab](https://developer.puter.com/labs/firefox-wasm/)

page, with TCP-over-WebSockets for networking and an experimental WASM-to-JS
JIT. Impractical by design, but a strong signal of where WASM's ceiling is
for porting large native codebases.

A multi-decade effort concludes: the FreeBSD base system is now entirely
