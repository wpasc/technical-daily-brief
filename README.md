# AI News Site

AI-generated news articles from open-source RSS feeds, powered by Claude.

## Overview

This project scrapes news from public RSS feeds, uses Claude to transform raw content into well-written articles, and displays them on a newspaper-style React frontend.

## Architecture

```
RSS Feeds --> Scraper --> Events DB --> Story Writer --> Articles DB --> Frontend
                                              |
                                          Claude API
```

- **Scraper**: Fetches RSS feeds from major news outlets (BBC, NPR, Guardian, etc.)
- **Story Writer**: Uses Claude to transform raw events into polished articles with different writer personas
- **Backend**: FastAPI REST API with SQLite database
- **Frontend**: React app with newspaper-style layout

## Quick Start

### Prerequisites

- Python 3.12+
- Node.js 18+
- uv (Python package manager)
- Anthropic API key

### Setup

1. Clone and navigate to the project:
   ```bash
   cd projects/news_site
   ```

2. Copy environment file and add your API key:
   ```bash
   cp .env.example .env
   # Edit .env and add your ANTHROPIC_API_KEY
   ```

3. Install dependencies:
   ```bash
   make setup
   ```

4. Initialize the database:
   ```bash
   source .venv/bin/activate
   cd backend && PYTHONPATH=. python -c "from core.database.session import init_db; init_db()"
   ```

### Running

**Option 1: Docker (recommended)**
```bash
make dev
```

**Option 2: Manual**

Terminal 1 - Backend:
```bash
source .venv/bin/activate
cd backend && PYTHONPATH=. uvicorn api.main:app --reload --port 8000
```

Terminal 2 - Frontend:
```bash
cd frontend && npm start
```

### Generating Content

1. Scrape news from RSS feeds:
   ```bash
   source .venv/bin/activate
   python scrapers/run_scraper.py
   ```

2. Generate articles with Claude:
   ```bash
   source .venv/bin/activate
   python story_writer/run_writer.py
   ```

3. View at http://localhost:3000

## Project Structure

```
news_site/
  backend/
    api/              # FastAPI routes
    core/
      models/         # SQLAlchemy models
      repositories/   # Data access layer
      schemas/        # Pydantic schemas
      database/       # DB session management
    lib/config/       # Configuration
  scrapers/           # RSS scrapers
  story_writer/       # LLM article generation
  frontend/           # React app
  data/               # SQLite database
```

## Writer Personas

Articles are generated by one of five AI writer personas:

- **Alex Chen** - Analytical Reporter: Data-driven, factual analysis
- **Sarah Mitchell** - Investigative Correspondent: Deep-dive investigations
- **Marcus Webb** - Tech Analyst: Technology and science coverage
- **Elena Rodriguez** - Human Interest Reporter: People-focused stories
- **David Park** - Commentary Writer: Opinion and analysis

## API Reference

| Endpoint | Method | Description |
|----------|--------|-------------|
| /api/health | GET | Health check |
| /api/events | GET | List events |
| /api/events | POST | Create event |
| /api/events/{id} | PATCH | Update event |
| /api/articles | GET | List articles |
| /api/articles | POST | Create article |
| /api/writers | GET | List personas |
